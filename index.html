My Works

Blog Posting Platform

I developed a dynamic and user-friendly blog platform using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The website allows users to create, read, update, and delete blog posts with ease. With a clean and responsive design, it offers features like user authentication, category-based content filtering, and real-time content updates. The back-end is powered by Node.js and Express.js, ensuring seamless API handling, while MongoDB stores user data and posts securely. React.js and Tailwind CSS handle the front-end, providing a smooth and engaging user experience.

Edu learn

EduLearn is an interactive online learning platform built using React.js, designed to offer a seamless and engaging educational experience. The platform provides users with access to a variety of courses, lessons, and learning resources. It features a user-friendly interface with dynamic components, allowing learners to easily navigate through course materials, track progress, and participate in quizzes. Leveraging React's state management, EduLearn ensures smooth content delivery and real-time updates. The platform is fully responsive, making it accessible across all devices, and offers an optimized learning experience for students and educators alike.

Yoga Connect

Yoga Connect is a holistic platform for yoga and mindfulness, designed to connect users with expert instructors and personalized classes. Built using the MERN stack (MongoDB, Express.js, React.js, and Node.js), the platform offers a smooth and intuitive user experience, enabling users to browse and enroll in yoga and meditation classes, manage schedules, and make secure payments via Stripe integration. With role-based functionalities, admins can manage instructors and classes, while users can track their progress and explore wellness content. Yoga Connect is crafted with Tailwind CSS for a clean and responsive design, providing users with a seamless and calming interface.
